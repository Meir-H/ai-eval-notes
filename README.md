# AI Eval Notes

Working notes on evaluating AI systems, written publicly as I develop them.

I build AI products and the frameworks that evaluate them. Most of my current
work is in adversarial evaluation for AI systems operating in safety-critical
domains, particularly healthcare. This repo is where the methodology thinking
lives separately from any specific project: the ideas, distinctions, and
mistakes that come up while building evaluation infrastructure for AI that
people actually rely on.

Nothing here is final. These are notes, not papers. If something in here
turns out to be wrong, the version that replaces it will say so.
